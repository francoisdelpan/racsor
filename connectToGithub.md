# Connexion a GitHub

Procedure simple pour creer ou configurer un compte GitHub.

## 1. Ouvrir GitHub

Aller sur :

https://github.com/

Si besoin, creer un compte ici :

https://github.com/signup

## 2. Renseigner l'adresse mail

Utiliser obligatoirement l'adresse mail professionnelle :

`@franchise.carrefour.com`

Exemple :

`prenom_nom@franchise.carrefour.com`

## 3. Choisir le nom d'utilisateur

Utiliser ce format :

`prenomnom-mm-carrefour`

Exemple :

`jeandupont-mm-carrefour`

## 4. Finaliser l'inscription

Suivre les etapes demandees par GitHub :

- validation de l'adresse mail
- choix du mot de passe
- connexion au compte
